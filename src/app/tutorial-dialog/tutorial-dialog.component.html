First steps using this APP

  1. To start, import or create a file.
  2. Before editing you must select file to operate with by selecting. (selected file name is shown in bold)
  3. To select a file, either click on its name in the tree visualization or click the menu button next to its name and select "Show file."
  4. To add or edit a visualization, right-click on the map with your mouse.
  5. To select a menu visualization, such as a track, move the mouse over it until the color changes to red, and then click the right button on the mouse.
  6. To add a waypoint, select "add waypoint" and click on the spot where you opened the menu.
  7. To create a track or route, the first point is automatically set to the place where the menu was opened. Additional points can be added by clicking on the map with the left button on the mouse.
  8. To edit settings for waypoints, tracks, or routes, expand the visualization in the tree and search for the specific element you want to edit.
  9. Click on the menu button next to the name of the element you choose and select "Settings" from the menu.
  10. Edit the settings as desired and click "Save" to apply the changes.
